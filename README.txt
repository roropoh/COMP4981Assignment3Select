lmsn is a linux chat application run through terminal.
==================================================================================================================================

Running the Application:
    1. The application can be run from terminal from the root directory

    run as client:
    2. ./lmsn client [username] [server ip] [server port]

    run as server:
    2. ./lmsn server [server port]



Compiling the project:
    - The project can be compiled using gcc or using the Makefile in /src
